## 简介
一个带日历的todo，目前是没有做移动端的适配，不过后续有时间会跟进。实现上主要用了React，配合moment.js做了一个日历组件。用leancloud做了数据的存储，实现你注册个号，哪儿都能用。其实todo部分很快就做出来了，主要的时间花在边上那个日历组件。最开始用原生js去得到日期，虽然可以实现但是，代码很难懂，自己都会维护晕掉。。。所以还是用了库，把日期的处理交给库。
